# UltraNavigator-Lethal-Company-mini-map-mod
---
  🧭 UltraNavigator

  ✨ Возможности

  🗺️ Радар реального времени

  Круговой радар с компасной розой (N/E/S/W) в левом нижнем углу экрана. Показывает всё, что происходит вокруг:
  
🔵 Корабль — всегда знай, где твоя база
🟢 Главный выход — зелёная метка к спасению
🟠 Запасные выходы — оранжевые точки к каждому выходу
🔴 Монстры — красные точки в реальном времени (подключается через Harmony-патч, реагирует на спавн и смерть врагов)
🟡 Лут — яркие жёлтые точки с белым свечением (ценный лот >60 — крупный оранжевый)
⚪ Другие игроки — белые точки

  ▎ 🎯 Близкие предметы рисуются очень крупно — не затеряются даже под ногами!

  🧭 Компас

  В левом верхнем углу:
  
⬆ Стрелка к главному выходу / кораблю с точной дистанцией (в метрах)
⬆ Стрелка к запасному выходу — отдельная строка с оранжевой стрелкой
💰 Счётчик лута — 💰 18 (1420$) — количество предметов и общая стоимость всего scrap'а на карте
👾 Счётчик монстров — сколько врагов рядом

  🚶 Путь на NavMesh

  Зелёные и оранжевые линии прямо на полу ведут к выходам. Работает через Unity NavMesh — пути обходят стены и
  препятствия. Дальность отрисовки — 40 метров.

  ⚙️ Настройки (BepInEx config)

  
🔘 Клавиша включения (по умолчанию N)
📏 Размер радара (80–300 px)
📐 Дальность радара (10–200 юнитов)
🟢 Вкл/выкл: компас, радар, линии пути, лут на радаре
🟠 Отдельное вкл/выкл путей к запасным выходам

  🧩 Технические детали

  
Canvas overlay (ScreenSpaceOverlay, sorting order 200) — не зависит от игрового HUD, рендерится поверх всего
Harmony-патчи: EntranceTeleport.Awake (ловим входы при спавне), EnemyAI.Awake/KillEnemy (монстры в реальном времени)
Обновление радара — ~10 FPS (каждый 6-й кадр) для производительности
Совместимость с v81

  ---
  📥 Установка

  
Скачай UltraNavigator.dll
Положи в BepInEx/plugins/UltraNavigator/
Запусти Lethal company
